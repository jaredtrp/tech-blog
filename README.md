## Description

# Usage

- Sequelize
- Handlebars
- Node
- SQL
- bcrypt

## Installation Instructions

1. npm init -y
2. npm i
3. npm i express-handlebars
4. npm i express-session connect-session-sequelize
5. npm i --save mysql2
6. npm i dotenv bcrypt
